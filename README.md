# ABOUT

ãƒ»A simple timer extention.  
ãƒ»When you start the timer, remaining time will be displayed on your browser until the timer expires.  
ãƒ»The timer runs every active tab. This mean that you can retaine the timer **even if reload or move to another tab.**

# Get started!

### `ãƒ»Download`

Access the [release page](https://github.com/mox692/ChromeExtention_SettingTimer/releases) and download the project folder.

```
$ git clone https://github.com/mox692/ChromeExtention_SettingTimer/
```
